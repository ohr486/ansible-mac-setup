# プロジェクト概要

これは、macOS環境をセットアップするためのAnsibleプレイブックです。Homebrewを使用してさまざまなパッケージやアプリケーションをインストールし、事前定義された一連のロールでシステムを構成します。

メインのプレイブックは`local.yml`で、実行するロールとインストールするパッケージを定義します。 `hosts`ファイルは、プレイブックを`localhost`で実行するように指定します。

`setup.sh`スクリプトは、依存関係（Xcodeコマンドラインツール、Homebrew、Ansible）のインストールとプレイブックの実行プロセスを自動化します。

# ビルドと実行

このプレイブックを実行するには、GitとHomebrewがインストールされている必要があります。次に,リポジトリをクローンして`setup.sh`スクリプトを実行します:

```bash
git clone git@github.com:ohr486/ansible-mac-setup.git
cd ansible-mac-setup
./setup.sh
```

`setup.sh`スクリプトは次のことを行います:

1. Xcodeコマンドラインツールをインストールします。
2. Homebrewをインストールします。
3. Ansibleをインストールします。
4. `local.yml`プレイブックで`ansible-playbook`コマンドを実行します。

# 開発規約

## プレイブックの構造

プレイブックはロールに整理されており、`roles`ディレクトリにあります。各ロールには、実行するタスクを定義する`main.yml`ファイルを含む`tasks`ディレクトリがあります。

## パッケージ管理

パッケージはHomebrewとHomebrew Caskを使用して管理されます。インストールするパッケージのリストは、`local.yml`ファイルに`brew_packs`および`brew_cask_packs`として定義されています。

パッケージを追加、削除、または更新するには、`local.yml`の適切なリストを変更する必要があります。

## カスタマイズ

プレイブックは`local.yml`ファイルを変更することでカスタマイズできます。ロールを追加または削除したり、インストールするパッケージを追加、削除、または更新したりできます。
